Extract zipFilePath to extractDir.

Sets chmod permissions for each file from the zip to the extracted one, which the built-in python zipfile module doesn't do by default. Source: Adapted from https://bitbucket.org/mumak/distribute/src/2f489b41a507/setuptools/archive_util.py
